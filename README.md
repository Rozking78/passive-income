| Daily | $1,429 |

## Quick Start

```bash
# Clone the repo
git clone https://github.com/Rozking78/passive-income.git
cd passive-income

# Run the system (no dependencies needed!)
python main.py
```

## Usage

### Interactive Mode
```bash
python main.py
```

### Command Line
```bash
python main.py research    # Research niches
python main.py content     # Generate content ideas
python main.py stats       # View dashboard
python main.py links       # Manage affiliate links
```

## Features

### 1. Niche Finder
- Analyzes trending topics from Google Trends
- Scans Reddit for active discussions
- Matches niches to high-paying affiliate programs
- Generates content gap ideas

### 2. Content Engine
- Video hooks and titles (YouTube, TikTok)
- Short-form video scripts (TikTok, Reels)
- Blog post outlines (SEO-optimized)
- Email sequences (4-email series)
- Social media post templates

### 3. Link Tracker
- Track unlimited affiliate links
- Record clicks by platform
- Log conversions and revenue
- Calculate conversion rates

### 4. Analytics Dashboard
- Progress toward $10k/week goal
- Top performing links
- Platform breakdown
- Revenue projections

## Strategy

### Phase 1: Foundation (Week 1-2)
- [ ] Research 5 profitable niches
- [ ] Sign up for 3-5 affiliate programs
- [ ] Create first 10 pieces of content
- [ ] Set up link tracking

### Phase 2: Content Machine (Week 3-8)
- [ ] Post daily on TikTok/Reels
- [ ] Publish 2 blog posts/week
- [ ] Build email list
- [ ] Engage with communities

### Phase 3: Optimization (Week 9-12)
- [ ] Analyze top performers
- [ ] Double down on winners
- [ ] Cut underperformers
- [ ] Scale ad spend (if profitable)

### Phase 4: Scale (Month 4+)
- [ ] Automate content creation
- [ ] Hire VA for engagement
- [ ] Expand to new platforms
- [ ] Launch own products

## The Math

To hit $10k/week with affiliate marketing:

| Strategy | Avg Commission | Sales Needed/Week |
|----------|----------------|-------------------|
| Low-ticket only | $20 | 500 ❌ |
| Mid-ticket | $100 | 100 |
| High-ticket | $300 | 34 |
| Recurring stack | $50/mo | 200 subscribers ✓ |

**Best approach:** Stack recurring commissions (SaaS tools) with occasional high-ticket sales.
